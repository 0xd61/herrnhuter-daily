Die Losungen

Losungen des Andachtsbuches der Herrnhuter Brüdergemeinde für Ihre Website

  • Dokumentation

REST-API Dokumentation

Dieses Projekt bietet Losungen als REST-API zur einfachen Einbindung in Websites oder Web Applications an. Die Losungen entsprechen der von der Evangelische Brüder-Unität – Herrnhuter Brüdergemeine ausgewählten Bibelverse und nutzt serverseitig die hier angebotenen Losungen im XML Format.

Die API dient als universelle Schnittstelle und bietet die Bibelverse im JSON-Format (JavaScript Object Notation) an.
Bei jeder Darstellung eines Bibeltextes muss eine Quellenangabe ersitlich sein, welche bei einem Request als Link zurückgegeben wird.

Das Projekt ist als Open Source Projekt auf Github öffentlich verfügbar.
Wir freuen uns auf Deine Mitarbeit! ;-)
